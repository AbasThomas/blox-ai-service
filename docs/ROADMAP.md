# Blox Roadmap (Phase 0-12)

- Phase 0: monorepo + infra + baseline CI.
- Phase 1: onboarding and create->publish core funnel.
- Phase 2: OAuth imports + profile unification.
- Phase 3: editor depth + ATS/SEO scans.
- Phase 4: billing/subscriptions and paywalls.
- Phase 5: realtime collaboration and branching.
- Phase 6: analytics and gamification.
- Phase 7: marketplace and e-commerce.
- Phase 8: scanner/duplicator/critique hardening.
- Phase 9: advanced AI (translation, smart sections, theme gen).
- Phase 10: coach/jobs/network/interview.
- Phase 11: enterprise admin and white-label.
- Phase 12: scaling, reliability, launch optimization.
